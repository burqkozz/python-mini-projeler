# Python Mini Projelerim

Bu repository, Python öğrenirken geliştirdiğim başlangıç seviyesi mini projeleri içerir.

## Projeler

* Alan ve Çevre Hesaplama
* Bütçe Takip Uygulaması
* Görev Takip Uygulaması
* Günlük Uygulaması
* Hesap Makinesi
* Kütüphane Sistemi
* Market Stok Sistemi
* Mini Quiz
* Not Defteri
* Not Hesaplama
* Öğrenci Kayıt Sistemi
* Sayı Kontrol Uygulaması
* Telefon Rehberi

## Kullanılan Konular

* Değişkenler
* Koşul yapıları

* Fonksiyonlar

* Sözlükler
* Dosya işlemleri

## Amaç

Bu projeler Python temel konularını öğrenmek, algoritma mantığını geliştirmek ve farklı konular üzerinde pratik yapmak amacıyla geliştirilmiştir.

Projelerde kullanıcıdan veri alma, dosya işlemleri, JSON ile veri kaydetme, liste ve sözlük yapıları, fonksiyon kullanımı, koşul yapıları ve döngüler gibi temel Python konuları uygulanmıştır.
